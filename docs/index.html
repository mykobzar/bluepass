Web installer requires Chrome or Edge (Web Serial API).
Download firmware for manual flashing via esptool.py.

HTTPS is required to use the web installer.
Open the secure version or download firmware.

Requires Chrome or Edge  ·  Web Serial API

After flashing
1
Device boots as WiFi access point bluepass (open network)
2
Connect to it and open http://192.168.4.1
3
Enter your WiFi credentials → Save & Reboot
4
Short-press GPIO0 on the device to activate the web UI on your local network
5
Go to Settings → Board to configure the correct button and LED pins for your specific ESP32-S3 board
