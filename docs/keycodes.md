## Modifier bits

Modifiers are combined as a bitmask in the `<Mod>` prefix.

| Bit | Name |
|-----|------|
| `0x01` | Ctrl / LCtrl |
| `0x02` | Shift / LShift |
| `0x04` | Alt / LAlt |
| `0x08` | Win / LWin |
| `0x10` | RCtrl |
| `0x20` | RShift |
| `0x40` | RAlt |
| `0x80` | RWin |
